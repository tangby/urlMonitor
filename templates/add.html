Create New Monitor
name 请输入一个名字（数字或下划线组合）
interval 请输入一个时间(单位为秒), 系统会按照这个间隔运行此监控,
mailInterval 请输入一个时间(单位为分钟), 系统在这个间隔内只发送一次email警报给你,
smsInterval 请输入一个时间(单位为分钟), 系统在这个间隔内只发送一次短信警报给你,
smsStartTime 请输入一个时间(如00:00), 系统在这个时间以后发送短信警报给你,
smsEndTime 请输入一个时间(如23:59), 系统在这个时间以后不发送短信警报给你,
times4rpt 请输入一个数字，系统将在此monitor连续失败n次以后发送警报,
interval4rpt 请输入重试间隔，单位为分钟,
description 请对此monitor做简单的描述
url 要监控的URL
timeout 请输入一个超时时间 单位是毫秒
contacts 报警联系人
statusCode 监控状态码
